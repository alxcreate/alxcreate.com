### Task Sequences

```text
рџ’Ѕ Task Sequences
    рџ“Ѓ OSD (OS deployments)
        рџ’Ѕ <Manufacturer> <OS><Edition> <Configuration> <Language> <Architecture>
    рџ“Ѓ OSI (.wim creating)
        рџ’Ѕ OSI <Manufacturer> <OS><Edition> <Configuration> <Language> <Architecture>
    рџ“Ѓ OSU (Windows updates (feature upgrade))
        рџ’Ѕ <Manufacturer> <OS><Edition> <Configuration> <Language> <Architecture>
    рџ“Ѓ Dev (Test)
        рџ’Ѕ DEV <Manufacturer> <OS><Edition> <Configuration> <Language> <Architecture>
```

Remove spaces and underscores in Manufacturer and Product values. These characters are used as delimiters.

- **Manufacturer** - Manufacturer. Do not use different spellings of the same manufacturer. If not specified or Opensource, "Others" is used.
- **Product** - Product name (Office, Photoshop, etc.). The marketing version number "2019" or "X" is written together.
- **Version** - Product version number in the format 0.0.0.0, 0.0.0 or 0.0. Do not use the marketing version number "2019" or "X" etc.
- **P** or **V** - Physical and virtual packages (APP-V) The type is specified in the deployment types.
- **Language** - Two-character language: RU, EN, DE, FR, etc.
- **Architecture** - Architecture (arm64/x64/x86/x99).
- **Configuration** - Configuration description is used when necessary.
- **OS** - OS (Windows7, Windows11, etc.).
- **Edition** - OS edition (Home, Pro, Ent, Edu, Mlt)
